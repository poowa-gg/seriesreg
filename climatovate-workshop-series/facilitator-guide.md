# Facilitator Guide

## Pre-Workshop Checklist (1 Week Before)

### Content Preparation
- [ ] Customize slide deck with month's topic and state data
- [ ] Confirm guest speaker and brief them on format
- [ ] Prepare hyperlocal data for featured states
- [ ] Create/update registration Google Form
- [ ] Set up post-session survey link
- [ ] Prepare 1-page handout PDF

### Technical Setup
- [ ] Schedule Zoom/Google Meet with recording enabled
- [ ] Test screen sharing and audio
- [ ] Prepare backup internet connection
- [ ] Create shortened links (bit.ly) for all resources

### Promotion
- [ ] Send email to mailing list
- [ ] Post on all social channels
- [ ] Share in WhatsApp community
- [ ] Send reminder 3 days before

---

## Day-Of Checklist

### 30 Minutes Before
- [ ] Open Zoom/Meet room
- [ ] Start recording
- [ ] Share waiting room message
- [ ] Test audio/video
- [ ] Have slides ready
- [ ] Open chat for monitoring

### During Session
- [ ] Welcome in English + Pidgin
- [ ] Remind about recording
- [ ] Monitor chat for questions
- [ ] Keep to time (60-90 mins)
- [ ] Run interactive poll
- [ ] Share survey link before closing

## Post-Workshop Tasks

### Same Day
- [ ] Stop recording and download
- [ ] Send thank-you email with recording link
- [ ] Post thank-you on social media
- [ ] Export survey responses

### Within 3 Days
- [ ] Upload recording to YouTube (unlisted)
- [ ] Create 3-5 short clips for social media
- [ ] Design summary infographic
- [ ] Share resources in WhatsApp group

### Within 1 Week
- [ ] Analyze survey data
- [ ] Update tracking spreadsheet
- [ ] Begin promotion for next month
